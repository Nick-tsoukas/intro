How to get access token for the Instagram feed through the Wordpress admin

  • 1. login into your Wordpress site
  • 2. Go to the dashboard
  • 3. Click on Instagram feed located on the left sidebar towards the bottom
  • 4. Click on the blue button to get access token and user ID
    • if you are logged in already Wordpress will generate token and id
    • if you are not logged in, you will be redirected to instagram to sign in and click the authorize button. Then you will be redirected back to the Wordpress admin with your token and id
